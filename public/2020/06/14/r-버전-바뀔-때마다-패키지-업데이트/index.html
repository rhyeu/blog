R 버전 업데이트

## install.packages("installr")
library(installr)
check.for.updates.R() # R 버전 업데이트 할 필요 있는 지 확인
updateR() # R 버전 업데이트

패키지 업데이트

update.packages(checkBuilt = T) # 모든 패키지 업데이트
packageStatus() # package 업데이트 확인
